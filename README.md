# Muzzakshare
Using Graphql and Create React App

### Required
Node 9.10.0

### Frontend
[Create React App](https://github.com/facebook/create-react-app)


### Backend
Backend using prisma as datebase and graphql generator.
You need to install it before
```
npm i -g prisma
```
